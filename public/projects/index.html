Projects

mlbcal

https://github.com/joeeoj/mlbcal

A pip-installable project that is a CLI wrapper around the MLB Stats API schedule endpoint.

Example usage:

$ mlbcal seattle --csv --nopre > mariners_schedule.csv

lahnman-to-duckdb

https://github.com/joeeoj/lahnman-to-duckdb

A wrapper around the baseballdatabank (aka Lahnman database) to convert the CSV files into a DuckDB database for faster, local analytics of baseball data. DuckDB is kind of like the OLAP version of SQLite.

Example queries against the database can be found here: examples.ipynb

recipes

http://whatsfordinner.recipes/

A static website, hosted by Netlify, with my favorite recipes and some editorial notes. Great for last minute dinner planning. Based on Recipes by jeffThompson with modifications by kvpsky.
